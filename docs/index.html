<teiCrafter> About Research Preview

A browser editor for TEI-XML editions

teiCrafter opens an existing TEI edition, shows it page by page next to its facsimile, and lets you correct the text and annotate entities with authority identifiers (GND, GeoNames, Wikidata). It runs in the browser; the edition stays a local file on your computer.

What saving does

Saving writes your edits into the original file text and changes nothing else: whitespace, attribute order, comments and entity notation outside the edited passages remain untouched. A document saved without edits is identical to the opened file.

What the editing unit is

Words where the edition encodes <w>, lines otherwise; the editor reads this from the document, no configuration is needed. Image zones and text are linked through @facs in both directions.

What is checked

While you edit, the editor checks that the XML stays well-formed and that no text content is lost, and reports the result next to the text. How machine-generated content is handled is described in About.
