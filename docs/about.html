About teiCrafter

teiCrafter is a browser-based, lossless editor for arbitrary TEI-XML. Open an existing edition, correct it folio by folio, annotate persons, places, organisations and works with authority data (GND, Wikidata, GeoNames), and save it back. It runs entirely in your browser: no server, no upload, no account. Your edition never leaves your computer.

Lossless, defined

The opened XML file is the single source of truth. Every edit is a precise splice into that raw text; everything you did not touch is preserved byte for byte, including whitespace, attribute order, comments and entities. Saving an unchanged document reproduces the input exactly. This property is verified continuously: live checks in the editor (the well-formed, lossless chip) and an offline test harness that round-trips hundreds of real edition files byte-identically.

Natural granularity

The editable unit emerges from the document instead of a forced profile: word-level when the TEI encodes words (<w>), line-level otherwise (<lb>/<l>). Page images travel along: the facsimile pane reads <facsimile>/<zone> and links text and image both ways via @facs.

AI assists, the human decides

Machine proposals never blend into human work: AI-origin content is marked violet and counted as unreviewed until a human verifies it; violet appears on AI-origin content and nothing else. An optional on-ramp that lets a language model draft an initial TEI from plaintext exists in the codebase and is switched off in this preview.

The example editions

Three real cases drive the tool: the Wenzelsbibel (word-level; the real Codex 2759 with IIIF facsimile where present, a synthetic stand-in for the licence-restricted codex in the public preview), a Jeanne Hersch letter from the Zentralbibliothek Zürich pipeline (line-level, page scans), and a letter from Stefan Zweig Digital (line-level, facsimile via the GAMS repository).

Method and provenance

teiCrafter is developed with Promptotyping: an AI-assisted, knowledge-document-driven development method in which every capability is backed by a verifiable proof. The repository, including the full knowledge base and test harness, is open: github.com/DigitalHumanitiesCraft/teiCrafter.
